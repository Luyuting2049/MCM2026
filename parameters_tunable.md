## 各模块输入参数（用户可调）

### 1. 显示屏模块 (Display)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| brightness | 浮点数 [0.0-1.0] | 0.5 | 屏幕亮度比例 |

### 2. 内存模块 (Memory)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| bw_r | 浮点数 ≥0 (GB/s) | - | 读取带宽 |
| bw_w | 浮点数 ≥0 (GB/s) | - | 写入带宽 |

### 3. 存储模块 (Storage)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| is_active | 布尔值 | False | 是否有I/O操作 |

### 4. 传感器模块 (Sensors)

| 参数 | 可选值 | 默认值 | 说明 |
|------|--------|--------|------|
| gps_mode | 'off', 'track', 'search' | 'off' | GPS工作模式 |
| cam_mode | 'off', 'preview', 'video' | 'off' | 摄像头工作模式 |
| imu_mode | 'low', 'high' | 'low' | IMU采样频率 |

### 5. 处理器模块 (Processor)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| f_cpu | 浮点数 ≥0 (MHz) | - | CPU工作频率 |
| mu_cpu | 浮点数 [0.0-1.0] | - | CPU利用率 |
| f_gpu | 浮点数 ≥0 (MHz) | - | GPU工作频率 |
| mu_gpu | 浮点数 [0.0-1.0] | - | GPU利用率 |

### 6. 蓝牙模块 (Bluetooth)

| 参数 | 可选值 | 默认值 | 说明 |
|------|--------|--------|------|
| mode | 'ble', 'classic' | 'ble' | 蓝牙模式 |
| class_type | '1', '2', '3' | '2' | 传统蓝牙类别 |
| duty_cycle | 浮点数 [0.0-1.0] | 0.1 | 发送占空比 |

### 7. 蜂窝网络模块 (Cellular)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| network_type | '2G', '3G', '4G' | '4G' | 网络类型 |
| rssi_db | 整数 [-100, -50] | -70 | 信号强度 |
| handoff_rate | 浮点数 ≥0 | 1 | 基站切换频率 |

### 8. WiFi模块 (WiFi)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| is_on | 布尔值 | True | WiFi是否开启 |
| rssi_db | 整数 [-100, -50] | -70 | 信号强度 |
| freq | '2.4GHz', '5GHz' | '2.4GHz' | WiFi频段 |
| is_scanning | 布尔值 | False | 是否在扫描网络 |

### 9. 热点模块 (Hotspot)

| 参数 | 类型/范围 | 默认值 | 说明 |
|------|-----------|--------|------|
| is_active | 布尔值 | True | 热点是否开启 |
| cellular_current | 浮点数 ≥0 | 70.0 | 蜂窝网络当前电流 |
| freq | '2.4GHz', '5GHz' | '2.4GHz' | 热点频段 |
| device_count | 整数 ≥1 | 1 | 连接的设备数量 |
